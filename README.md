# Stack
- Front-end: Angular
- Back-end: Node.js
- API: MongoDB

## Rodando o projeto
- O .env deve ter "MONGODB_URI" e "PORT"
- Dentro da pasta da API rode "node --watch server.js"
- Dentro da pasta do front end rode "ng serve --open"
